## Langkah Ajaib: Netlify Drop

Kita akan menggunakan layanan bernama **Netlify**. Ini adalah "Malaikat Penolong" para web developer modern.

### Step 1: Rapikan Folder

Pastikan file kodinganmu (misal `kalkulator.html`) diganti namanya menjadi **`index.html`**.

Kenapa? Karena internet sudah sepakat bahwa "pintu gerbang" sebuah website itu namanya harus `index.html`.

Masukkan file itu ke dalam satu folder khusus, misalnya folder bernama `Project_Kalkulator`.

> **TIPS: Dimana Folder Saya?**
>
> Kalau kamu bingung file-nya tersimpan di mana, tanya saja ke AI:
>
> *"Di mana lokasi folder proyek ini di laptop saya? Tolong buka foldernya."*
>
> AI akan memberitahu jalurnya atau bahkan ngebukain File Explorer buat kamu.

### Step 2: Buka Netlify Drop

Buka browser, ketik: **[app.netlify.com/drop](https://app.netlify.com/drop)**.

Kamu akan melihat area kotak putus-putus bertuliskan *"Drag and drop your site folder here"*.

![Tampilan Netlify Drop](/blog/antigravity/netlify_drop_tutorial.png)

### Step 3: Seret dan Lepas

Buka folder `Project_Kalkulator` kamu di komputer, **klik tahan (drag)** foldernya, lalu **lepas (drop)** ke kotak di website Netlify tadi.

Tunggu 5 detik...

*Uploading...*

*Published!* 🎉

### Step 4: Ganti Nama (Optional)

Websitemu sudah online! Tapi namanya biasanya aneh, misal: `bambang-ganteng-x7z89.netlify.app`.

Klik **"Site Settings"** → **"Change Site Name"**.

Ubah jadi nama yang keren, misal: `kalkulator-pajak-budi.netlify.app`.

## FAQ: "Gratisnya Sampai Kapan?" (PENTING)

Banyak yang tanya: *"Mas, ini beneran gratis? Ada limitnya gak?"*

Jawabannya: **Gratis Selamanya**, tapi dengan sistem **Kuota (Bandwidth)**.

**Logika Gampangnya:**

Bayangkan Netlify itu seperti **Provider Internet**.

- Kamu dikasih **Kuota Data 100 GB / bulan**.
- Setiap kali ada orang yang buka websitemu, kuotanya berkurang sedikit (paling cuma beberapa Kilobyte).
- **100 GB itu Berapa Banyak?** Itu SANGAT BANYAK.
- Kalau websitemu cuma Kalkulator sederhana, butuh **Jutaan Pengunjung** buat ngabisin kuota itu dalam sebulan.

**Kesimpulan:** Selama websitemu bukan sekelas Tokopedia atau Facebook, kuota gratis ini **RASANYA SEPERTI UNLIMITED**. Aman!

## 🎁 Bonus Chapter: The Antigravity Prompt Cheat Sheet

**"Contekan Mantra" untuk Memerintah AI**

Ini adalah "Senjata Rahasia" Anda. Tidak perlu hapal coding, cukup copy-paste mantra ini untuk membuat AI bekerja 10x lebih efektif.

---

### 1. Mantra "Arsitek" (Untuk Memulai Project)

**Gunakan saat:** Baru mau bikin aplikasi dari nol.

> "Halo, saya ingin membuat aplikasi berbasis web untuk **[Tujuan Aplikasi, misal: kasir warung]**.
>
> Tolong berikan saya **struktur folder** dan **file apa saja** yang dibutuhkan.
>
> Gunakan HTML, CSS, dan JS vanilla saja agar mudah. Buat desainnya modern dan responsif untuk HP."

### 2. Mantra "Tukang Dekor" (Untuk Mempercantik Tampilan)

**Gunakan saat:** Tampilan aplikasi masih kaku seperti tahun 1990.

> "Tolong ubah desain file CSS-nya. Saya ingin gayanya **'Modern & Clean'**.
>
> Gunakan font 'Inter' atau 'Poppins' dari Google Fonts.
>
> Berikan efek bayangan (box-shadow) lembut pada kartu dan tombol.
>
> Pastikan tombolnya enak dipencet di layar HP (padding besar)."

### 3. Mantra "Satpam Logic" (Untuk Mencegah Error)

**Gunakan saat:** Ada hitungan matematika yang salah atau error aneh.

> "Coba cek lagi logika hitungan di fungsi `hitungTotal`.
>
> Pastikan kamu **[Jelaskan aturan, misal: tidak boleh ada angka negatif]**.
>
> Buatkan saya fungsi 'Guardian' untuk memfilter input yang aneh-aneh.
>
> Jika ada error, tolong jelaskan kenapa, dan berikan kode perbaikannya secara lengkap (jangan potong-potong)."

### 4. Mantra "Detektif Error" (Saat Mentok)

**Gunakan saat:** Aplikasi tidak jalan, layar putih, atau console merah semua.

> "Saya dapat error ini di Console: **[Paste Error Disini]**.
>
> Tolong analisa penyebabnya. Jelaskan kepada saya seperti menjelaskan ke anak umur 5 tahun.
>
> Lalu perbaiki kodenya. Saya tidak mahu tahu teknisnya, pokoknya harus jalan."

### 5. Mantra "Deploy" (Saat Mau Pamer)

**Gunakan saat:** Project sudah jadi dan mau di-online-kan.

> "Project saya sudah selesai. Semua file ada di folder ini.
>
> Pastikan file utamanya bernama `index.html`.
>
> Apakah ada file gambar atau script yang path-nya salah (broken link)? Tolong cek dan perbaiki sebelum saya upload ke Netlify."
